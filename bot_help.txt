/help - выводит справочную информацию по командам
/start - регистрирует пользователя

/new <session_name> - создает новую сессию для работы с отдельным изображением
/use <session_name>|<session_id> - меняет текущую сессию на другую
/list - список сессий

/select <feature_id> - показать признак на фото
/overlay_show <overlay_name> - показать накладываемое изображение
/overlay_list - список накладываемых изображений
/pos_list - список позиций для накладываемого изображения

/overlay <feature_id> <overlay_name> [<position> [<scale>]] - наложить изображение <overlay_name> на признак <feature_id> в положение <position> с масштабом <scale>

/show - показать итоговый результат по сессии
/close - закрыть сессию
